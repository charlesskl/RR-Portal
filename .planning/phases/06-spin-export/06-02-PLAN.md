修改 `server/routes/export.js`:

2. 在 router.get handler 中，查询 version 后根据 format_type 分支:

```javascript
let buffer;
if (version.format_type === 'spin') {
  buffer = await exportSpinVersion(req.params.versionId);
} else {
  buffer = await exportVersion(req.params.versionId);
}
```

3. 文件名也根据格式区分:
```javascript
const prefix = version.format_type === 'spin' ? 'SPIN_VQ' : 'VQ';

```

前端无需改动 — 现有的 `api.exportExcel(versionId)` 调用 GET `/api/export/:versionId`，路由会自动根据 format_type 分支。
