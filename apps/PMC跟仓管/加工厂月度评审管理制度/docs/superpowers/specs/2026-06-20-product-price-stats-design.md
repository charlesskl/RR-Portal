# 产品单价统计表 — 设计文档

日期:2026-06-20
状态:已确认,待实现

## 背景与目标

新增一个「产品单价统计表」板块,可导出 Excel,格式对齐桌面参考文件
`啤机部-外发产品单价统计表.xlsx`。该表按 车间 → 加工厂 → 加工类别 分组,
每行是一个产品(货号 + 配件名称/模号)及其价格信息。

参考文件结构(单 sheet,标题 + 两级表头):

```
车间 | 加工厂名称 | 加工类别 | 货号 | 配件名称/模号 | [价格管理: 核价生产工价$ | 供应商外发价￥ | 外发单价$ | 扣税点1.13后单价$ | 占比] | 综合评价(经理评分) | 备注
```

左侧「车间 / 加工厂名称 / 加工类别」三列对同组数据用合并单元格。

## 数据来源

由现有 **orders(下单明细)** 数据生成,每条订单 = 一个产品价格行。不新建独立集合。

### orders 需新增 2 个字段

| 字段 | 名称 | 类型 |
|------|------|------|
| `supplier_price` | 供应商外发价￥ | number |
| `process_category` | 加工类别 | text |

### 已复用的现有字段

| 统计表列 | orders 字段 |
|----------|-------------|
| 车间 | `workshop` |
| 加工厂名称 | `factory.name`(expand) |
| 货号 | `item_no` |
| 配件名称/模号 | `product` |
| 核价生产工价 | `quote_labor_price` |
| 外发单价 | `unit_price` |
| 扣税点1.13后单价 | 计算 = `unit_price / 1.13` |
| 占比 | 计算 = 扣税点1.13后单价 / `quote_labor_price` |
| 经理评分 | `manager_rating` |
| 备注 | `notes` |

## 计算口径(与参考文件一致,已用样本验证)

- 扣税点1.13后单价 = 外发单价 ÷ 1.13(保留 4 位小数)
- 占比 = 扣税点1.13后单价 ÷ 核价生产工价(百分比,保留 1 位小数)
  - 验证:0.58/1.13 = 0.5133;0.5133/0.525 = 97.8% ✓
  - 核价为 0/空 或外发单价为空 → 占比显示「-」

## 新字段录入位置

- **新增下单表单**(`OrderFormView.vue`):增加「供应商外发价」「加工类别」输入。
- **订单详情页**(`OrderDetailView.vue`):新增可编辑的「价格信息」区,
  集中编辑 核价生产工价 / 外发单价 / 供应商外发价 / 加工类别(保存写库)。
- **不**加入下单明细宽表(`DeptOrdersView.vue`,已 15 列),避免过宽。

## 导航与页面

- 侧边栏(`AppLayout.vue`)新增菜单「产品单价统计」。
- 路由:
  - `/price-stats` → 部门卡片页(注塑/喷油/装配/车缝),仅显示有数据的部门(与 `/orders` 一致)。
  - `/price-stats/dept/:craft` → 某部门的统计表页。
- 统计表页:
  - 按 车间 → 加工厂 → 加工类别 分组排序的**只读表格**,左侧三列用 `rowspan` 合并同组。
  - 右上角「导出 Excel」按钮。
  - 复用现有 orders store;按 `expand.factory.craft === craft` 过滤本部门订单。

## Excel 导出格式(照参考文件)

- 标题行:`{部门}-外发产品单价统计表`,横向合并所有列。
- 两级表头:
  - 第一级:车间 / 加工厂名称 / 加工类别 / 货号 / 配件名称/模号(各跨 2 行)
    + 「价格管理」(横跨 5 个价格列)+ 综合评价 / 备注。
  - 第二级:核价生产工价$ / 供应商外发价￥ / 外发单价$ / 扣税点1.13后单价$ / 占比 / 经理评分(1-10分)。
- 数据行按 车间→加工厂→加工类别 排序,**车间 / 加工厂 / 加工类别 纵向合并**同组单元格。
- 列宽按内容自适应(沿用 SummaryView 的列宽算法)。
- 文件名:`{部门}-外发产品单价统计表.xlsx`。

## 不做(YAGNI)

- 不做独立价格目录/主数据(已确认由订单生成)。
- 不做跨部门合并大表(按部门)。
- 不把新字段塞进下单明细宽表。
- 不在本期做价格历史/版本。

## 边界与注意

- 分组合并:仅纵向合并**连续相同**的 车间/加工厂/加工类别;排序须保证同组相邻。
- 占比/扣税点为计算列,不入库。
- 空数据时导出仍输出标题 + 两级表头。
- 权限:菜单与页面默认所有登录用户可见(与下单明细一致);如需限制后续再加。
