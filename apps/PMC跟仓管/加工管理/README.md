# 唱片机管理系统

PMC 跟仓管下的唱片机物料流转系统，用于按部门记录 NFC 贴纸、PCBA 板的出入、领料、成品/半成品入库、供应商资料和公开汇总。

## 功能

- 登录时选择部门，管理员 `admin` 可进入所有部门，普通账号只能进入所属部门。
- 部门包含：兴信B来料仓、装配、半成品、外发、河源华兴、邵阳、新邵。
- 兴信B来料仓支持供应商资料维护，并在出入记录中记录供应商。
- 邵阳和新邵成品入库记录支持 PO 和客名。
- 各页面支持日期和单据编号筛选。
- 登录页提供公开汇总入口，未登录也可只读查看汇总详情。
- 所有用户可修改自己的密码，管理员可管理用户和重置密码。

## 安装

```bash
python -m pip install -r requirements.txt
```

## 启动

```bash
python -m uvicorn pcba.main:app --host 0.0.0.0 --port 8000
```

- 本机访问：http://localhost:8000
- 云端平台路径：`/cpg/`

## 默认账号

- 管理员：`admin` / `admin123`

## 备份

数据默认存放在 `data/pcba.db`，定期复制该文件即可备份。
