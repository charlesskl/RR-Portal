界面结构说明

  • 顶部导航栏：幽砚紫主色，显示系统名称 + 用户角色
  • 左侧菜单：啤机部 / 印喷部 / 装配部 切换 + 三工汇总
  • 工具栏：日期筛选、快捷按钮（近7天/本月/本季）、搜索、新增/导入/导出/批量删除
  • 数据表格：中间可滚动区域，表头紫色固定
  • 底部固定合计：各车间合计 → 清溪合计 → 邵阳合计 → 总合计，始终可见
  • 结余列：用橄榄绿(#3D8361)高亮，一眼看出核心数据
