# 汇总表改版设计 — 按日汇总 + 按月汇总

> 日期：2026-03-23
> 状态：已确认
> 目标：替换现有汇总表，为未来利润核算（生意额 - 人工成本 - 物料成本 - 固定支出 + 三工结余）提供清晰的三工数据源

## 1. 变更范围

替换"汇总表"标签页内容。"可视化看板"标签页保持不变。

**现有结构（删除）：** 4 个子标签（总览 / 啤机部 / 印喷部 / 装配部），行=费用分类，列=车间/部门

**新结构：** 2 个子标签

| 子标签 | 数据范围 | 筛选条件 |
|--------|---------|---------|
| 按日汇总 | 单部门，按日展开 + 月度合计 | 部门 + 月份 + 跳转日期 |
| 按月汇总 | 三部门合并月度对比 | 月份 |

## 2. 按日汇总

### 2.1 筛选栏

| 控件 | 类型 | 说明 |
|------|------|------|
| 部门 | 三选一按钮组（啤机部/印喷部/装配部） | 默认啤机部 |
| 月份 | el-date-picker type="month" | 默认当月 |
| 跳转日期 | el-date-picker type="date" | 选择后页面自动滚动到对应日卡，限制在所选月份范围内 |
| 导出 Excel | 按钮 | 导出当前部门当月全部数据 |

### 2.2 月度合计卡（置顶）

- **位置：** 筛选栏下方，日期卡片上方
- **头部颜色：** 橄榄绿渐变（#3D8361 → #2d6b4e），边框 2px solid #3D8361
- **头部信息：** "[月份]合计 · [部门名]月度汇总" + 总产值 / 总结余 / 结余率
- **表格结构：** 行=车间，列=该部门所有费用字段
- **列顺序：** 车间（sticky固定）→ 总产值 → 所有 expense:true 字段（按 config.js 定义顺序：共有费用 → 独有费用）→ 费用合计 → 结余 → 结余率
- **末行：** 合计行，背景 #edf7f2，字体加粗，顶部 2px solid #3D8361 分隔线
- **数据来源：** 当月所有记录按车间 SUM

### 2.3 日期卡片（倒序排列）

- **分隔标记：** 月度合计卡与日卡之间用虚线分隔，中间标注"▼ 每日明细"
- **头部颜色：** 浅雾紫渐变（#D391DD → #c080cc）
- **头部信息：** "YYYY-MM-DD · 周X" + 产值合计 / 结余合计
- **表格结构：** 与月度合计卡完全同构（行=车间，列=所有费用字段）
- **表头背景：** #f9f0fb（淡紫），表头文字颜色 #7F41C0
- **合计行背景：** #f9f0fb，顶部 1px solid #D391DD 分隔线
- **排序：** 按日期倒序（最新日期在最上方）
- **数据来源：** 单日原始记录

## 3. 按月汇总

### 3.1 筛选栏

| 控件 | 类型 | 说明 |
|------|------|------|
| 月份 | el-date-picker type="month" | 默认当月 |
| 导出 Excel | 按钮 | 导出三部门月度汇总 |

### 3.2 本月三工汇总表（上半区）

- **标题：** "📊 本月三工汇总（YYYY年M月）"，底部 2px solid #7F41C0 分隔线
- **表头：** 背景 #7F41C0，白色文字
- **表格结构：** 行=三个部门 + 三工合计行，列为共有费用大类归拢

| 列 | 说明 |
|----|------|
| 部门 | 啤机部 / 印喷部 / 装配部 |
| 总产值 | SUM(daily_output) |
| 员工工资 | SUM(worker_wage) |
| 管工工资 | SUM(supervisor_wage) |
| 房租 | SUM(rent) |
| 水电费 | SUM(utility_fee) |
| 社保 | SUM(social_insurance) |
| 税收 | SUM(tax) |
| 其他费用 | 部门独有费用之和 + 剩余共有费用之和。剩余共有费用 = tool_investment(工具投资) + equipment(设备) + renovation(装修) + misc_fee(杂费) + shipping_fee(运费) |
| 费用合计 | 所有 expense:true 字段之和 |
| 结余 | 总产值 - 费用合计 |
| 结余率 | 结余 / 总产值 |

- **三工合计行：** 背景 #f5f0fa，字体加粗 700，顶部 2px solid #7F41C0
- **单位：** 万元，右下角标注

### 3.3 环比变化表（下半区）

- **标题：** "📈 环比变化（vs YYYY年M月）"，底部 2px solid #5B9BD5 分隔线
- **表头：** 背景 #5B9BD5，白色文字
- **表格结构：**

| 列 | 说明 |
|----|------|
| 部门 | 三部门 + 三工合计 |
| 总产值变化 | ↑/↓ 差值 (+百分比%) |
| 费用合计变化 | 同上 |
| 结余变化 | 同上 |
| 结余率变化 | ↑/↓ 百分点 |
| 上月结余 | 上月结余金额 |
| 本月结余 | 本月结余金额 |

- **颜色语义：** 有利变化（产值↑/费用↓/结余↑）用 #3D8361，不利变化用 #E88EA0
- **注意：** 费用变化中"↑"是不利的（花更多），"↓"是有利的（花更少），颜色语义与产值/结余相反
- **三工合计行：** 背景 #EBF3FB，字体加粗 700

## 5. 后端 API 变更

### 5.1 新增/修改接口

**GET `/api/summary/daily?dept=beer&month=2026-03`**

后端解析 `month` 参数为日期范围：`WHERE record_date >= '2026-03-01' AND record_date < '2026-04-01'`

返回按日汇总数据：
```json
{
  "monthly": {
    "workshops": [
      { "name": "兴信A", "daily_output": 52.8, "worker_wage": 18.2, ... , "total_expense": 39.4, "balance": 13.4, "balance_ratio": 0.254 },
      ...
    ],
    "total": { "daily_output": 186.5, ... }
  },
  "daily": [
    {
      "date": "2026-03-03",
      "weekday": "周二",
      "workshops": [ ... ],
      "total": { ... }
    },
    ...
  ]
}
```

**GET `/api/summary/monthly?month=2026-03`**

返回按月汇总数据：
```json
{
  "current": {
    "departments": [
      { "dept": "beer", "label": "啤机部", "daily_output": 186.5, "worker_wage": 62.3, ... , "other_expense": 38.7, "total_expense": 138.8, "balance": 47.7, "balance_ratio": 0.256 },
      ...
    ],
    "total": { ... }
  },
  "comparison": {
    "departments": [
      { "dept": "beer", "label": "啤机部", "output_change": 12.3, "output_change_pct": 0.071, "expense_change": 8.5, ... , "prev_balance": 43.9, "curr_balance": 47.7 },
      ...
    ],
    "total": { ... }
  }
}
```

### 5.2 保留接口

- `GET /api/summary/dashboard` — 可视化看板数据，不变
- `GET /api/summary/overview` — 当前无前端消费者，暂保留，后续可清理

### 5.3 可删除接口

- `GET /api/summary/detail` — 被 daily + monthly 替代
