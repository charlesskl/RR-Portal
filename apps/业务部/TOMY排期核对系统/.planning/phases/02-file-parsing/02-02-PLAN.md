Create `server/lib/excelParser.ts`:

1. **Import ExcelJS and types:**
   ```typescript
   import ExcelJS from 'exceljs'
   import type { ScheduleRow } from '../types/index.js'
   ```
   If `server/types/index.ts` does not exist yet (parallel wave), define ScheduleRow locally and re-export.

2. **buildColumnMap(headerRow)**: Read row 1, map each cell's trimmed string value to its column number. Return `Map<string, number>`.

3. **COLUMN_ALIASES** constant (from research):
   ```typescript
   const COLUMN_ALIASES: Record<string, string[]> = {
     tomyPO: ['Tomy PO', 'TOMY PO'],
     customerPO: ['Cust. PO NO.', 'CUSTOMER PO'],
   }
   ```
   For Chinese column names (国家, 第三客户名称, etc.), use direct lookup since they are identical in both files.

4. **resolveColumn(colMap, names)**: Try each alias in order, return first matching column index or undefined.

5. **unwrapCellValue(raw)**: Handle three cases:
   - `null/undefined` -> `null`
   - Formula cell (`typeof raw === 'object' && raw !== null && 'result' in raw`) -> extract `.result`
   - Normal value -> return as-is
   This is CRITICAL for 外箱, 总箱数, 数量 columns which are often formula cells.

6. **parseScheduleExcel(buffer)**: Main export function.
   - `const workbook = new ExcelJS.Workbook(); await workbook.xlsx.load(buffer)`
   - Get worksheet named '总排期', throw Error if not found
   - Build column map from row 1
   - Iterate rows starting from row 2 (skip header)
   - For each row, extract all ScheduleRow fields using column map + aliases
   - Handle Date cells: check `value instanceof Date`, keep as Date object
   - Handle formula cells: use unwrapCellValue
   - Convert tomyPO and customerPO to trimmed strings (they may be numbers in Excel)
   - Skip rows where tomyPO is empty/null (empty rows in spreadsheet)
   - Return ScheduleRow[]

7. **IMPORTANT from research:**
   - Do NOT use column index constants — always look up by header name
   - Do NOT call `.toString()` on cells — returns "[object Object]" for formulas
   - Date cells in PO走货期 and 接单期 columns return JS Date objects
