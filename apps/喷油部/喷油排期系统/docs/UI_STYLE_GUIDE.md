## 0. 风格基调（业务方 2026-05-25 拍板）

| 维度 | 决策 | 业务方原话 |
|---|---|---|
| 整体气质 | **薄荷柔和**（Linear / Notion 风）| "倾向小清新" |
| 留白 | **大**（卡片间距 16-24px、模块间距 36px）| "大片留白多" |
| 按钮形状 | **中圆角 12px**（介于药片 999px 和方形 4px 之间）| "药片风格" |
| 按钮颜色 | **分色鲜明**（每个动作类型独立色 + 阴影）| "按钮颜色分别鲜明" |
| 阴影 | 柔和淡薄（卡片）+ 色相阴影（按钮）| 截图气质 |
| 字体 | PingFang SC / Microsoft YaHei / Inter | 默认无衬线 |

## 6. 布局规范

### 6.1 整体框架

```
┌───────────────────────────────────────────────┐
│  顶部导航 (white, 64px, sticky)               │
│  • 印喷部 V1   首页 订单 排期 ...    🟢 admin │
├───────────────────────────────────────────────┤
│                                               │
│  ┌────── 主内容区（max-width 1480px）──────┐  │
│  │  background: #f7f9fb                   │  │
│  │  padding: 32px                         │  │
│  │                                        │  │
│  │  [页面标题 + 副标题 + 右上日期]         │  │
│  │  [提示横幅 💡（如有）]                   │  │
│  │  [4 列统计卡片]                         │  │
│  │  [区块标题 + 3 列功能卡片]              │  │
│  │  [列表片段（含分色按钮 + 筛选 + 表格）] │  │
│  └────────────────────────────────────────┘  │
│                                               │
└───────────────────────────────────────────────┘
```

### 6.2 顶部导航

- 高 64px，白底，sticky 固定
- 左侧：品牌"🟢 印喷部 V1 系统"（薄荷绿小圆点 + 粗体）
- 中部：菜单项，间距 4px，padding 8/16，圆角 10px
  - 默认：`color #6b7280`
  - 悬停：`bg #f3f4f6, color #1f2937`
  - active：`bg #ecfdf5, color #047857, font-weight 600`
- 右侧：圆头像（薄荷绿 + 白字）+ 用户名（admin 小灰字角色）+ "登出" 白底按钮

### 6.3 Dashboard 页面

#### 6.3.1 标题区
```
左：H1 "印喷部 控制台"
    副标题 "欢迎回来，<strong>系统管理员</strong> · admin"

右：日期 "2026-05-25 / 周一" 右对齐灰字
```

#### 6.3.2 提示横幅
- 薄荷淡底 `#ecfdf5` + 左边 3px 薄荷绿竖线 + `💡` emoji
- 字色 `#065f46`，13px

#### 6.3.3 统计卡片（4 列）

| 元素 | 样式 |
|---|---|
| 容器 | grid 4 列，gap 16px |
| 卡片 | 白底，圆角 16px，padding 22px，border `1px solid #f1f5f9` |
| Label | 小圆点（多色）+ 13px 灰字 |
| 数字 | 36px 700 `#0f172a` |
| Hint | 12px `#9ca3af` |

**统计卡片小圆点配色（4 个固定指标）：**
| 卡片 | 圆点色 | 指标 |
|---|---|---|
| 1 | `#34d399` 薄荷绿 | 今日计划生产数（件）|
| 2 | `#60a5fa` 天蓝 | 本月在产订单（张）|
| 3 | `#a78bfa` 浅紫 | 本月完结订单（张）|
| 4 | `#f87171` 暖红 | 已逾期订单（张）|

#### 6.3.4 功能卡片 grid（3 列）
- gap 16px
- 每张：图标方块（40px 圆角 12px 彩色底）+ chips + H3 标题 + 描述
- 选中态：边框 `2px dashed #34d399` + 底色 `#f0fdf4`

### 6.4 列表页

- 整张列表用白卡片包裹（圆角 16px，padding 24px）
- 顶部：左侧标题（带 4px 薄荷绿左竖线）；右侧分色按钮组
- 筛选区：浅灰底 `#f9fafb` 内嵌卡片，grid 布局，输入框 + 主按钮
- 表格：表头薄荷淡底 `#f0fdf4`，表头字 `#047857`，行 hover `#fafdfb`

## 7. 组件规范

### 7.1 按钮（更新）

```css
/* 通用 */
border: none;
padding: 10px 16px;
border-radius: 12px;
font-size: 13px;
font-weight: 600;
cursor: pointer;
display: inline-flex;
align-items: center;
gap: 6px;
transition: all 0.15s;

/* 分色见 § 1.3.3 */
/* 阴影按色相，hover translateY(-1px) */
```
