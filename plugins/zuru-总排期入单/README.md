# ZURU 总排期入单（云端分析版）

解析 ZURU PO Excel → 匹配总排期 → 输出修改单明细 + 生成新单 Excel。

## 功能

- 上传总排期 xlsx 副本（用于匹配）
- 批量上传 PO Excel → 自动识别修改单/新单
- 修改单：展示行号+字段变化明细，供人工参考更新
- 新单：生成独立 Excel（按分排期 sheet 分组）供下载粘贴
- 重复 PO 自动去重（同 PO 保留最新版本）
- 货号黑名单过滤（`data/ignore_items.json`）
- 有填充行汇总（扫描整行填充，按分排期分类统计）
- 分排期归属查找

## 云端 vs 本地

| 特性 | 云端分析版（本插件） | 本地完整版 |
|------|-------------------|-----------|
| PO 解析 + 修改单识别 | ✅ | ✅ |
| 新单 Excel 生成 | ✅ | ✅ |
| 有填充行汇总 | ✅ | ✅ |
| WPS COM 写入总排期 | ❌（Linux 无 WPS） | ✅ |
| Z 盘直接读写 | ❌（需上传副本） | ✅ |
